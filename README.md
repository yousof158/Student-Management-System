## 📝 Project Description

A clean and interactive Java console application that implements a custom `BinarySearchArray` data structure. The array automatically maintains a sorted order upon insertion and utilizes a highly efficient **Recursive Binary Search** algorithm to find elements. Additionally, the system tracks the number of **comparisons and swaps** made during operations to help analyze algorithm performance and efficiency.

## ✨ Features

*   **Auto-Sorting Insertion:** Elements are placed in their correct sorted position immediately upon insertion, maintaining a strictly ordered array.
*   **Recursive Binary Search:** Fast and efficient searching utilizing the divide-and-conquer approach to quickly locate elements.
*   **Element Deletion:** Seamlessly removes elements and shifts the remaining data to maintain the array's sorted structure without leaving gaps.
*   **Performance Metrics Tracking:** Accurately counts and displays the total number of comparisons and swaps executed during the program's lifecycle.
*   **Interactive CLI:** A user-friendly Command Line Interface (CLI) that guides the user through inserting, searching, and deleting elements step-by-step.

## 🛠️ Technologies Used

*   **Language:** Java
*   **Core Concepts:** Object-Oriented Programming (OOP), Data Structures (Arrays), Algorithms (Binary Search, Insertion Sort Logic), Time Complexity Tracking.

## 💡 Usage Example

When you run the application, the console will prompt you to:
1. Enter the total number of elements you want to insert.
2. Input each element one by one (the program will auto-sort them).
3. Search for a specific value using the recursive binary search.
4. Optionally delete any value from the array.
5. Finally, view the total algorithmic cost (Comparisons & Swaps) for all operations performed.
